🪖 יצירת טופס גיוס לפלוגה

⚠️ חשוב: קובץ זה מכיל הוראות מפורטות ליצירת טופס Google Forms עבור גיוס חיילים למערכת ניהול הפלוגה.

🎯 מטרת הטופס

הטופס ישמש לקליטת מועמדים חדשים לפלוגה. הנתונים יועברו אוטומטית למערכת הניהול לשיבוץ ואישור.

📋 שלב 1: יצירת הטופס

1. היכנס ל-Google Forms: https://forms.google.com
2. לחץ על "+" ליצירת טופס חדש
3. שנה את הכותרת ל: "טופס גיוס - פלוגה לוחמת"
4. הוסף תיאור: "מלא את הטופס הבא להצטרפות לפלוגה"

📝 שלב 2: הוספת השדות

הוסף את השדות הבאים בדיוק בסדר הזה:

פרטים אישיים (חובה)

1. כתובת אימייל
סוג: Text - Short answer
✅ סמן כשדה חובה
✅ הפעל אימות אימייל
2. שם מלא
סוג: Text - Short answer
✅ סמן כשדה חובה
3. מספר אישי
סוג: Text - Short answer
✅ סמן כשדה חובה
תיאור: "9 ספרות ללא רווחים"
4. מספר טלפון
סוג: Text - Short answer
תיאור: "לדוגמה: 050-1234567"

פרטים נוספים

5. תאריך לידה - סוג: Date
6. כתובת מגורים - סוג: Text - Short answer
7. איש קשר לחירום - סוג: Text - Short answer
8. טלפון חירום - סוג: Text - Short answer

רקע ומיומנויות

9. השכלה
סוג: Multiple choice
אפשרויות: תיכון, תעודת הנדסאי, תואר ראשון, תואר שני, אחר
10. שפות
סוג: Checkboxes
אפשרויות: עברית, אנגלית, ערבית, רוסית, צרפתית, אחר
11. רקע צבאי
סוג: Text - Paragraph
תיאור: "ציין יחידות קודמות, קורסים, הסמכות"
12. פרופיל רפואי
סוג: Multiple choice
אפשרויות: 97, 82, 72, 64, 45, 21, אחר

מוטיבציה והתאמה

13. מוטיבציה לשירות
סוג: Text - Paragraph
תיאור: "מדוע אתה רוצה לשרת ביחידה?"
14. ציפיות מהשירות
סוג: Text - Paragraph
תיאור: "מה אתה מצפה ללמוד ולחוות?"
15. תחומי עניין/תחביבים - סוג: Text - Short answer
16. מידע נוסף
סוג: Text - Paragraph
תיאור: "כל מידע נוסף שחשוב לך לשתף"

⚙️ שלב 3: הגדרות הטופס

לחץ על הגדרות (⚙️) ובחר:
✅ אסוף כתובות אימייל
✅ הגבל לתשובה אחת
✅ דרוש כניסה עם Google

🔗 שלב 4: חיבור למערכת

1. עבור לכרטיסייה "תשובות"
2. לחץ על שלוש הנקודות ← "בחר יעד תשובה"
3. בחר "יצירת גיליון אלקטרוני חדש"
4. פתח את הגיליון ולחץ על הרחבות ← Apps Script

💻 שלב 5: הוספת הקוד

מחק את הקוד הקיים והוסף את הקוד הבא:

function onFormSubmit(e) { const formData = { email: e.values[1], // כתובת אימייל fullName: e.values[2], // שם מלא personalNumber: e.values[3], // מספר אישי phone: e.values[4], // טלפון birthDate: e.values[5], // תאריך לידה address: e.values[6], // כתובת emergencyContact: e.values[7], // איש קשר חירום emergencyPhone: e.values[8], // טלפון חירום education: e.values[9], // השכלה languages: e.values[10], // שפות militaryBackground: e.values[11], // רקע צבאי medicalProfile: e.values[12], // פרופיל רפואי motivation: e.values[13], // מוטיבציה expectations: e.values[14], // ציפיות hobbies: e.values[15], // תחביבים additionalInfo: e.values[16] // מידע נוסף }; // שליחה למערכת const functionUrl = 'https://us-central1-zikit-4c55c.cloudfunctions.net/processGoogleFormSubmission'; const options = { method: 'POST', contentType: 'application/json', payload: JSON.stringify(formData) }; try { const response = UrlFetchApp.fetch(functionUrl, options); Logger.log('תשובה מהמערכת: ' + response.getContentText()); } catch (error) { Logger.log('שגיאה בשליחה למערכת: ' + error.toString()); } }

🎬 שלב 6: הגדרת טריגר

1. שמור את הסקריפט (Ctrl+S)
2. לחץ על "טריגרים" בתפריט השמאלי
3. לחץ על "+ הוסף טריגר"
4. בחר:
   • פונקציה: onFormSubmit
   • סוג אירוע: הגשת טופס
5. שמור

🚀 שלב 7: פרסום הטופס

1. חזור לטופס Google Forms
2. לחץ על "שלח" בפינה הימנית העליונה
3. העתק את הקישור
4. שתף עם מועמדים לגיוס
🎯 תוצאה: כל מועמד שימלא את הטופס יופיע אוטומטיה במסך "חיילים ממתינים לשיבוץ" במערכת הניהול.
